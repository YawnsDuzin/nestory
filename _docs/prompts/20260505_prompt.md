====================

cli의 claude code 에서는 superpowers 가 인식이 되는데,, vs code extension 에서는 인식이 안되고 있어.

[추가1] 현재 프로젝트에도 D:\dzp\바이브코딩\investment-advisor\.claude\settings.json 와 같이 아래의 플러그인 설정해줘.

"enabledPlugins": {
    "context7@claude-plugins-official": true,
    "skill-creator@claude-plugins-official": true,
    "claude-code-setup@claude-plugins-official": true,
    "superpowers@claude-plugins-official": true
  }

[추가2] 그냥 1번으로 다 설치해줘
[추가3] 설치완료했어.
[추가4] D:\dzp\바이브코딩\investment-advisor\.claude\settings.json 와 현재 프로젝트의 superpowers 설정 차이가 무엇인지 확인해서 알려줘. 현재 프로젝트에서는 아직 안되는 것 같아.
[추가5] D:\dzp\바이브코딩\investment-advisor 프로젝트에서는 첨부이미지와 같이 /bra 치면 확인이 되는데, 현재 프로젝트에서는 /bra 처도 아무것도 표시가 안되고 있어.

$env:CLAUDE_PLUGIN_ROOT="C:\Users\dzP\.claude\plugins\cache\claude-plugins-official\superpowers\5.1.0"
& "$env:CLAUDE_PLUGIN_ROOT\hooks\run-hook.cmd" session-start
